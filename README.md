# web3-dev-starter

A comprehensive Web3 development starter template with Next.js frontend and blockchain integration.

## Project Structure

- `frontend/`: Next.js application with Web3 authentication
- `blockchain/`: Smart contract development environment (coming soon)

## Getting Started

1. Install dependencies:
```bash
npm run install:all
```

2. Start the development server:
```bash
npm run frontend:dev
```

Visit `http://localhost:3000` to see your application.

## Available Scripts

- `npm run frontend:dev` - Start the frontend development server
- `npm run frontend:build` - Build the frontend for production
- `npm run frontend:start` - Start the frontend production server
- `npm run frontend:install` - Install frontend dependencies
- `npm run install:all` - Install all dependencies
